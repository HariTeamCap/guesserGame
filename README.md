# 🎯 Guesser Game

A simple Java console-based game where a **Guesser** selects a number and three **Players** try to guess it. The **Umpire** collects all the guesses, compares them, and announces the winner.

This project demonstrates the fundamentals of **Object-Oriented Programming (OOP)** in Java.

---

## 📌 Features

- Console-based interactive game
- One Guesser and three Players
- Umpire controls the game flow
- Displays the winning player if the guess matches
- Demonstrates OOP concepts such as:
  - Classes & Objects
  - Encapsulation
  - Method Invocation
  - Object Collaboration

## 🚀 How It Works

1. The **Guesser** enters a number between **1 and 10**.
2. Three **Players** enter their guesses.
3. The **Umpire** compares all guesses.
4. If any player's guess matches the Guesser's number, that player wins.
5. Otherwise, the game displays that nobody guessed correctly.
